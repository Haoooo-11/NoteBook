#### §1.1 判断质数
[模板](https://leetcode.cn/problems/prime-in-diagonal/submissions/632334461/)
```python
def cheak(self, num):
    for i in range(2, isqrt(num) + 1):
        if num % i == 0:
            return False
    return num >= 2 # 1不是质数
```
即只要搞出在这个范围里遍历就行了:
\[ [2, \lfloor \sqrt{n} \rfloor] \]
